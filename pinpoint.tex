% An alignment system includes a laser transmitter and receiver. The receiver% includes a linear photodetector array centered at a predetermined location% with respect to machined surfaces that define a receiver reference plane. An% indicator on the receiver displays a deviation of the position of the laser% beam from a predetermined location on the photodetector array thereby to% quantify any misalignment of a surface supporting the receiver with respect% to the transmitted laser beam.O inventor Albert M. Creighton desenvolveu um sistema de alinhamento a laser queinclui um transmissor e receptor \citep{creighton2001laser}. O receptor possuiuma matriz de fotodetectores e um indicador mostra o desvio de posição do feixede laser do transmissor.
